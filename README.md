# 4scroll

This application can collect feeds of your favorite facebook pages and aggregate to one feed.

## To build and run a project

### `cd server`
### `npm install`
### `cd client`
### `npm install`
### `cd ..`
### `npm run dev`

sever\config\keys.js should be:

`module.exports = {
    googleClientID: "key",
    googleClientSecret: "key",
    mongoURI: "key",
    cookieKey: 'key',
    fbAccessKey: "key"
};`
